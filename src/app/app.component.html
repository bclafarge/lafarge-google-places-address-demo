What To Share With Another Team

Required

Google Maps JavaScript API key, Places API enabled, frontend loader, autocomplete input, and backend fields for address, latitude, and longitude.

Key Variants

Use an environment key, runtime config file, CI/CD injected variable, or backend-issued restricted config. Do not commit unrestricted production keys.

Other Languages

The same Google concepts apply outside Angular. See the JavaScript and React examples in the documentation folder.
